Abstract

Smart Campus Navigator is designed to improve on-campus mobility by combining building floor-plans, POI (departments, labs, canteen), and shortest-path routing. It supports search-by-room, accessibility options and admin-managed updates.

Project Modules

Map & Floor Plans

Upload and render multi-floor maps, toggle layers, zoom/pan and position markers.

Routing Engine

Graph-based pathfinding to compute shortest/accessible routes between points.

Search & POI

Search campus entities (rooms, faculty, services) with autocomplete and filtering.

Admin Panel

Admins can edit maps, add POIs, and push urgent notices to users.

User Profiles

Students save favorites, view history and set accessibility preferences.

Notifications

Real-time alerts (events, maintenance) and scheduled notifications for routes.
